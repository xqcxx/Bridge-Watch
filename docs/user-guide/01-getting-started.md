# Getting Started with Stellar Bridge Watch

## Welcome!

This guide will help you get started with Stellar Bridge Watch in just a few minutes.

## What You'll Learn

- How to access the platform
- Understanding the main interface
- Your first asset monitoring setup
- Creating your first alert

## Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- Basic understanding of blockchain and Stellar network
- (Optional) A Stellar wallet for advanced features

## Step 1: Access the Platform

1. Navigate to the Stellar Bridge Watch URL
2. The platform is publicly accessible - no login required for basic monitoring
3. For advanced features (alerts, preferences), you'll need to connect a wallet

## Step 2: Explore the Dashboard

When you first land on the platform, you'll see:

- **Asset Overview**: Real-time health scores for monitored assets
- **Bridge Status**: Current status of all monitored bridges
- **Recent Alerts**: Latest system alerts and notifications
- **Price Charts**: Live price data from multiple sources

## Step 3: Monitor Your First Asset

1. Click on any asset card (e.g., USDC, PYUSD)
2. View detailed information:
   - Current health score
   - Liquidity across DEXs
   - Price from multiple sources
   - Historical trends
3. Bookmark assets you want to track regularly

## Step 4: Set Up Your First Alert

1. Navigate to the Alerts page
2. Click "Create Alert"
3. Choose your alert type:
   - Price deviation
   - Bridge supply mismatch
   - Liquidity drop
   - Health score threshold
4. Configure your alert parameters
5. Add notification preferences (webhook, email)
6. Save your alert

## Step 5: Customize Your Experience

1. Go to Settings
2. Configure:
   - Display preferences
   - Notification settings
   - Data refresh intervals
   - Theme (light/dark mode)

- 💡 Hover over any metric to see detailed tooltips
- 💡 Click on charts to zoom in on specific time periods
- 💡 Use the search bar to quickly find assets or bridges

## Common First-Time Questions

**Q: Do I need to create an account?**
A: No account is required for basic monitoring. Connect a wallet for advanced features.

**Q: Is the data real-time?**
A: Yes, most data updates every 30 seconds. Some metrics update every 5 minutes.

**Q: Can I export data?**
A: Yes, use the Export feature to download data in CSV, JSON, or Excel format.

**Q: Is this free to use?**
A: Yes, Stellar Bridge Watch is completely free and open-source.
